4. DESIGN & UX REQUIREMENTS
--------------------------------------------------------------------------------
THEME:
- Dark mode default (modern DevOps aesthetic)
- Light mode toggle (accessibility)
- Color palette suggestion:
  * Primary:   Deep blue/navy (#1a237e) or AWS orange (#ff9900)
  * Secondary: Teal/cyan (#00bcd4) for accents
  * Dark bg:   #0f172a (slate-900)
  * Light bg:  #f8fafc (slate-50)
  * Text:      slate-100 on dark, slate-900 on light

TYPOGRAPHY:
- Clean, modern sans-serif (Inter, Geist, or Outfit)
- Mono font for code/tech terms (JetBrains Mono, Fira Code)
- Readable sizing, WCAG AA compliant contrast

RESPONSIVENESS:
- Mobile-first design
- Breakpoints: mobile <640px, tablet 640-1024px, desktop >1024px
- Hamburger menu on mobile
- Touch-friendly tap targets (min 44px)

PERFORMANCE:
- Lighthouse score: 90+ across all categories
- Core Web Vitals: LCP <2.5s, FID <100ms, CLS <0.1
- Optimize images (WebP, lazy loading)
- Minify JS/CSS
- Use CDN

ANIMATIONS:
- Subtle, purposeful animations only:
  * Fade-in on scroll (IntersectionObserver)
  * Card hover effects (scale/shadow)
  * Skill bar fill animation
- Respect prefers-reduced-motion

ACCESSIBILITY:
- WCAG 2.1 AA compliance
- Keyboard navigation
- ARIA labels
- Alt text on all images
- Focus indicators
